# Excel Sales Performance Dashboard

This project focuses on creating an interactive Excel dashboard to analyze sales performance across different countries, brands, industries, and sales representatives.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts

## Process
- Performed data cleaning by removing duplicates, correcting formats, and handling missing values.
- Applied sorting and filtering to organize and validate data.
- Created pivot tables to summarize sales value and sales count.
- Built an interactive dashboard using slicers for year, month, and region selection.
- Designed charts to visualize sales by country, brand, industry, and sales representatives.

## Key Insights
- Sales performance varies significantly across countries and regions.
- Certain brands and industries contribute higher sales value.
- Sales representatives show different performance patterns across time periods.

## Conclusion
This project demonstrates practical Excel skills in data cleaning, pivot table analysis, and dashboard creation for business reporting and decision-making.
